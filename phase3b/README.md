# المرحلة 3.2 (كامل) — قالب النشرة الكامل

القالب الآن كامل: الترويسة + الأخبار + الطقس + الصلاة + التوصيات + الفعاليات + التذييل (QR).
كله بيانات تجريبية، بنقوش نجدية SVG وأيقونات مرسومة.

> الصق كل سطر وحده. بلا شرطات مائلة.

## الخطوات

### 1) ادخل المجلد
```
cd ~/news-lebanon
```

### 2) انقل nashra-phase3b.zip عبر AnyDesk ثم فكّه
```
unzip -o nashra-phase3b.zip -d phase3b
```

### 3) انسخ القالب الكامل
```
cp phase3b/template/newsletter_preview.html src/newsletter_preview.html
```

### 4) ولّد الصورة
```
docker compose exec app php render_preview.php
```

### 5) افتح المعاينة
```
open src/storage/app/newsletter_preview.png
```

## ماذا تراجع
- الطقس: الأيقونات، العظمى/الصغرى، شريط الأيام الأربعة.
- الصلاة: الأيقونات الستة، الفواصل، «بتوقيت دار الفتوى».
- التوصيات/الفعاليات: البطاقات الموحّدة، الأيقونات الدائرية، الشارات.
- التذييل: رمز QR + العبارة + سلسلة النقوش.
- التناسق العام بين كل البلوكات.

## بعد المعاينة
أرسل لقطة أو ملاحظاتك. نضبط أي تفصيل، ثم ننتقل لـ3.3:
ربط القالب ببياناتك الحقيقية من قاعدة البيانات (بدل البيانات التجريبية).
